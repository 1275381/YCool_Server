#YCool_server

##前端地址

https://github.com/dlyt/YCool.git

##安装
```bash
git clone https://github.com/dlyt/YCool_Server.git
npm install
```
配置mongoodb,redis在YCool_Server/config/env/development.js

##开发
```bash
npm run dev
```

##API文档

```bash
npm run docs
```

运行项目后查看： http://localhost:5000/docs/
